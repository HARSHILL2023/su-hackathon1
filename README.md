# SmartFactory AI – TexTech Intelligence Hub

SmartFactory AI is a next-generation industrial monitoring and optimization platform specifically designed for the textile clusters in Bhilwara, Rajasthan. It leverages real-time data from factory edge nodes to provide actionable insights into production, compliance, and operational efficiency.

## 🚀 GovAssist AI: Our Newest Feature

The **GovAssist AI** module is a strategic intelligence layer designed to bridge the gap between complex factory operations and government regulatory systems. This module solves the critical challenges factory managers face when dealing with compliance reporting, inspection readiness, and subsidy applications.

### 💎 Key Modules in GovAssist AI

#### 1. AI Compliance Assistant
- **Automated Monitoring**: Real-time tracking of pollution, labor, and safety regulations.
- **Risk Scoring**: Live evaluation of compliance health (High/Medium/Low) based on actual production telemetry.
- **Deadline Management**: Visual tracking of upcoming regulatory filing dates.

#### 2. Subsidy Matcher (Siena AI Engine)
- **Data-Driven Matchmaking**: Automatically checks factory investment and energy data against real-world schemes like **TUFS** and **RIPS 2022**.
- **Official Submission Bridge**: One-click redirection to official government portals (MSME, Ministry of Textiles) after internal AI validation.

#### 3. Audit Simulation (Inspection Readiness)
- **Predictive Auditing**: Simulates surprise government inspections using machine health and safety data.
- **Action Items**: Generates prioritized tasks to fix hazards (e.g., lint accumulation, fire safety) before real inspectors arrive.

#### 4. Automated Industrial Reporting
- **1-Click Generation**: Automatically compiles complex regulatory documents (Environmental, Energy, Production reports) into high-fidelity PDFs using live factory data.
- **Zero-Manual Entry**: Eliminates human error in reporting for submission to industrial portals.

## 🛠️ Technology Stack
- **Frontend**: React.js with Tailwind CSS & Vanilla CSS (Futuristic Dark UI)
- **Backend**: Node.js & Express
- **Intelligence**: Rule-based AI Engine for industrial logic
- **Data Visualization**: Recharts
- **PDF Engine**: jsPDF & AutoTable

## 📦 Getting Started

### 1. Prerequisites
- Node.js (v14+)
- MongoDB (Running locally or via Atlas)

### 2. Installation
```powershell
# Clone the repository
git clone https://github.com/HARSHILL2023/su-hackathon1.git
cd su-hackathon1

# Install Dependencies
cd backend && npm install
cd ../frontend && npm install
```

### 3. Execution
```powershell
# Start Backend (Port 3001)
cd backend
node server.js

# Start Frontend (Port 3000)
cd frontend
npm start
```
